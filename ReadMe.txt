### GeocoderDemo ###

===========================================================================
DESCRIPTION:

Geocodeing Sample App that does basic forward string and dictionary geocodes and reverse coordinate geocodes. 
Also includes a distance between two placemarks calculator.

===========================================================================
BUILD REQUIREMENTS:

Xcode 4.2 or later, Mac OS X 10.6.6 or later. iOS 5.0 or later.

===========================================================================
RUNTIME REQUIREMENTS:

Mac OS X 10.6.6 or later. iOS 5.0 or later.
